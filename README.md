## 开发者打开前的准备工作

1. 你需要配合[配套的服务端程序](https://gitee.com/fx_wan/management-system-server)才可以正常使用。
2. 确保电脑上已经安装了yarn和vue-cli 4.5+。
3. 确保电脑上已经安装了git。

### 我的开发环境

- 系统：Windows 10 专业版 64位
- 硬件：Intel处理器 + 16GB内存 + 1TB固态
- 浏览器：微软Edge（Chrome内核）

## 开发者首次打开

1. ```
   yarn
   ```

   或

   ```
   yarn install
   ```
   
2. ```
   yarn dev
   ```
